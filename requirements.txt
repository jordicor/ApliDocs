# ApliDocs requires an SQLite provider with FTS5 and SQLite >= 3.53.3. The old
# pysqlite3-binary fallback is intentionally not installed: its latest
# 0.5.4.post2 wheel embeds SQLite 3.51.1, before the FTS5 memory-safety fixes.
#
# There is no single safe wheel compatible with every Synology CPU/glibc/Python
# combination. Probe the exact scheduled interpreter as documented in
# docs/SYNOLOGY.md. For read-only CLI clients, cysqlite is preferred where
# compatible; the audited CPython 3.12 win_amd64 cysqlite 0.3.4 wheel embeds
# SQLite 3.53.3, and runtime checks remain mandatory. The builder needs a
# current stdlib/pysqlite3-compatible provider from a vendor, container or
# self-built Python. The reproducible Windows client command is intentionally
# separate and pins cysqlite==0.3.4; see README.md. PDF extraction additionally
# requires `pdftotext`.
